# Job Board

#### By Will Johnson

## Description

A very simple C# assignment that allows users to post job objects to a page. It was mostly for the sake of getting a hang of how the language works.

## Setup/Installation Requirements

* Windows Machine
* Powershell

* Clone Project
* Run DNU Restore
* Run DNX Kestrel
